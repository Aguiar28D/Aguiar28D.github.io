House Price Prediction in Uruguay

Machine learning project focused on predicting house prices in Uruguay using real estate listings data from Mercado Libre and El Gallito (2023–2025). The analysis includes data cleaning, outlier treatment, feature engineering, and model comparison (Linear Regression vs CatBoost), with hyperparameter tuning using Optuna. Best performance is achieved for Montevideo, with a MAPE of approximately 25%.
